# statsd-wrap
DogStatsd wrapper lib for Python.

# Requirements
* Python `3.6` or `3.7`

## Install

## Development

**1. Install local dev depencies:**

```bash
pip install -e ".[dev]"
```

**2. Run tests:**
```bash
./run_tests.sh
```

**3. Make you changes, run tests again, commit and update version number on `setup.py` file.**

**4. Create new release on Github.**
